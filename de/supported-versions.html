Derzeit unterstützte Versionen

OpenMage LTS-Anwendern wird dringend empfohlen, immer die neueste Version zu verwenden. Frühere Versionen werden nur unterstützt, wenn Ressourcen verfügbar sind. Wenn Sie eine Version verwenden, die hier nicht aufgeführt ist, aktualisieren Sie bitte auf eine unterstützte Version.
