Currently Supported Versions

OpenMage LTS users are strongly encouraged to always follow the latest release. Past releases will only be supported as resources are available. If you are using a version that is not listed here, please update to a supported version.

OpenMage LTS Version Based on Magento Version Github Branch Initial Release Date
19.4.3 1.9.4.5 1.9.4.x 10 May 2020

OpenMage LTS is based on Magento Community Edition, tracking the latest official release until the EOL date on June 30, 2020.

The RFC for the long-term release process is currently in Draft state so participation from the community is appreciated and encouraged.
